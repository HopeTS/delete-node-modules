# delete-node-modules

CLI command for deleting node_modules folders in the current directory or a given path

## Installation

```bash
npm install @hopets/delete-node-modules
```

## Usage

```bash
# Delete node_modules folders recursively from current directory
delete-node-modules

# Delete node_modules folders recursively from given directory
delete-node-modules absolute/path/to/folder
```
